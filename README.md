# Task Viewer Beta

This is the initial version of a simple Windows Forms application developed in C# using Visual Studio. This project, *Task Viewer Beta*, serves as a practice tool for learning basic GUI development and integration with external processes.

## Features:

* *Custom Menu Strip:* A graphical menu interface for application navigation.
* *GitHub Integration:* Includes a menu option that opens the developer's GitHub profile in the default web browser (using System.Diagnostics.Process.Start).
* *Minimalistic UI:* A clean and simple user interface.

## Technologies Used:

* C#
* .NET / .NET Framework
* Windows Forms (WinForms)

## How to Get Started:

1.  *Clone the Repository:* Download the project files to your local machine.
2.  *Open in Visual Studio:* Open the .sln file located in the root directory.
3.  *Build and Run:* Compile the project and run the executable (.exe) file found in the bin/Debug folder.
